Всем привет!

1. Перейти по ссылки https://git-scm.com/
2. Справа найти и нажать кнопку "Download for Windows" 
3. В списке найти раздел "Standalone Installer" и кликнуть по ссылке которая соответствует архитектуре вашей ОС
3.1 Как проверить архитектуру вашей ОС

- Пуск
- Параметры
- О системе
- Найти раздел "Тип системы" (Пример: 64-разрядная операционная система, процессор x64)

4. Скачать подходящую версию
5. Установить git (во время установки выбираем только путь уставноки, остальное проклацываем по умолчанию)
6. Найти папку с git и в папке запустить git-bash.exe
7. Когда откроется "черное окошко с цветной надписью <user_name>@<computer_name> <os_architecture_type> - значит все сделано правильно

Теперь перейдем к более интересному

1. Для того чтобы создать ваш первый проект (папку) вам понадобится перейти в то место где находится гит
2. Пишем в баш-консоли следующее

2.1 cd <disc:/folder/folder>
2.2 Пример: cd D:/git
2.3 Нажимаем "Enter"
2.4 Если все сделано правильно ваша цветная строка преобразуется в что-то на подобе этого DartNova@DESKTOP-1HLR13P MINGW64 /d/git

! Обратите внимание что последнее в строке это место где уставнолен гит. В моем случае это диск "D" и папка "git"

3. Теперь пишем mkdir <название вашего проекта>
3.1 Жмем "Enter"
3.2 Теперь переходим в проект через команду из пункта 2.2 (ВАЖНО: Мы уже находимся в папке с гитом это значит что нам не надо занаво писать весь путь)
3.3 Поэтому команда должна быть cd <project_name> и нажимаем "Enter"

4. Теперь создаем репозиторий
4.1 Пишем команду git init <repository name> Пример: git init novytskyi
4.2 Enter
4.3 Переходим в репозиторий с помощью команды cd

5. Проверим что наш репозиторий пустой с помощью команды ls
5.1 Пишем ls и жмем Enter
5.2 Как результат будет пуста я строка (все верно ведь мы ничего в него еще не положили)

6. Пора добавить в нашу локальную репу что-нибудь
6.1 Создайте любой файл
6.2 И положите его в папку с вашей репой вручную (когда вы создавали проект и репу в баш-консоли то в вашей папке с гитом появлялись эти папки)
6.3 Теперь снова проверим что находится в нашей репе с помощью команды ls
6.4 Как видите теперь там лежит файл с именем который вы ему дали
6.5 Но нам нужно чтобы гит его УВИДЕЛ (чтобы понять видит он его или нет напишите в строке команду git status / если ваш файл красный
- то он его не видит а если зеленый то видит)

7. Открываем глаза гиту
7.1 Пишем команду git add <имя файла.расширение файла> и жмем Enter
7.2 Теперь проверяем видит его гит или нет через команду из пункта 6.5 (название файла должно стать зеленым)

8. Добавляем еще пару файлов в папку
8.1 Откроем глаза гиту на все файлы сразу через команду git add --all
8.2 Снова проверяем уидел ли гит ваши новые файлы

9. Теперь нужно перевести наши файлы в фазу "Staging Area" чтобы потом отправить их в интернет
9.1 Перевод в фазу делается через так называемый коммит
9.2 Пишем команду git commit -m "тут описываем что мы сделали до этого вкратце" и жмем Enter

Поздравляю мы подготовили наши файлы для отправки в мир а теперь нам надо сделать приготовления со стороны мира

10. Переходим по ссылке https://github.com/
10.1 В правом верхем углу находим кнопку "Sing up" и регистрируемся
10.2 После регистрации в левом углу находим кнопку "New" и жмакаем на нее
10.3 Далее в строке "Repository name *" вводим имя репозитория такое же как у вас на комухтере
10.4 Жмакаем кнопку "Create repository"
10.5 Теперь в голубой области убедитесь что у вас выбран "HTTPS"
10.6 Теперь правее от адреса вашего репозитория находим кнопку с двумя квадратиками и жмакаем на нее
10.7 Возвращаемся в вашу баш-консоль и пишем команду  git remote add origin ПРОБЕЛ и через контекстное меню вставляем ссылку из гит-хаба
10.8 Вот что должно получится git remote add origin https://github.com/DartNova/novytskyi.git
10.9 Жмакаем Enter
10.10 Теперь надо чтобы ветки в гит и в гит-хабе были одинаковые
10.11 Пишем команду git push --set-upstream origin master

11. Теперь пора отправить наши файлы из репозитория на компе в репозиторий на гит-хабе
11.1 Пишем git pull origin
11.2 Идем в гитхаб и находим вкладку "Repositories" и жмакаем на нее
11.3 Жмакаем на нашу репу
11.4 Проверяем что ваши файлы там
